# Карта не территория

Модель, план, описание — всегда упрощение реального устройства вещей. Ошибка
начинающего — спорить с реальностью, опираясь на модель, вместо того чтобы
проверить модель по реальности.

**Пример:** план недели — карта. Если пятница пошла не по плану, ошибка не у
пятницы — стоит пересмотреть карту.
